Description

Liquid monster is 2d platformer puzzle game. This game needs two player for playing this game. I mainly implemented debugging system so that designers could create various levels easily in short time. I created management system that saves the texture of all objects in advance to save loading time when the level changed. And I also developed and designed item components as gameplay programmer.
