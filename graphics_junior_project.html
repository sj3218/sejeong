Environment Mapping

This project shows Skybox and Environment Mapping(Reflection & Refraction).
There is a main object which is rendered with environment map on it. And the other objects are rotating around the main object.
There is UI for changing environment mapping: reflection, refraction and fresnel. In the refraction part, the user can change the refraction index from 1.0 to 100.0.
And also, there are pre-set materials for the refraction. For example, Air, Oil, Diamond and so on.

Deferred Rendering

I implemented hybrid rendering pipeline that employs a Deferred Rendering method for the main object, coupled with the usual Forward Rendering for auxiliary rendering(debug draw, vertex, and face normals etc). In the UI on the right side, User can see the individual render targets on Full Screen Quad.

I implemented three ways of Shading: Phong Lighting, Phong Shading, and Blinn Shading.
In Phong Lighting, Lightings computations are done in the vertex shader, and only the final color is interpolated to the fragment.
Phong Shading is similar with Phong Lighting but the lightings computations are implemented in the fragment shader.
Finally, Blinn Shading used half-vector between the light vector and view vector, and combine it with the surface normal for the specular computation. And the lighting computations are done in the framgent shader.

There are three lighting ways: Point light, Directional light and Spot light.
