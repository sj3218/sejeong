Description

Freeze Tag is VR Action single game using Unity engine. The main goal of this game is that gather all of the gems in the map and then release the NPCs while avoiding the enemy AIs.
I implemented Enemy AI using A* search algorithm for optimize AIs. I supported VR controller for hand simulation. I developed mini-map which detects position of the player and enemies for game UI and giving more realism in VR.
